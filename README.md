# -jrtejaswini--Predictive-Analysis-
House Price Prediction is a machine learning project that analyzes historical housing data to estimate property prices. The project involves data preprocessing, feature engineering, and predictive modeling using Linear Regression and Random Forest Regression. Model performance is evaluated using MAE, RMSE, and R² Score, while visualizations help interpret predictions and feature importance. The system can predict the price of a house based on factors such as living area, number of bedrooms, bathrooms, garage capacity, construction year, neighborhood, and overall quality.
